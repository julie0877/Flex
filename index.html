O que é display flex:

organizando elementos filhos automaticamente em uma linha ou coluna

facilita o alinhamento, distribuição de espaço e criação de layouts responsivos

O que é justify-content:

propriedade alinha os itens do contêiner flexível quando eles não utilizam todo o espaço disponível no eixo principal (horizontalmente)

define como o espaço livre é utilizado, permitindo posicionar elementos no início, fim, centro ou distribuí-los com espaçamentos uniformes

O que é aling-itens:

define como os itens (filhos) de um contêiner Flexbox ou Grid se alinham no eixo transversal (perpendicular ao eixo principal)

    Ela é usada para posicionar elementos verticalmente (quando em linha) ou horizontalmente
